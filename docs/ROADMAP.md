# StreamGuard Roadmap

## Phase 1 - MVP (Current)
- [x] Custom Commands with variables
- [x] Timers with chat-line threshold
- [x] Moderation filters (links, caps, symbols, emotes)
- [x] Twitch OAuth2 authentication
- [x] Dashboard with live chat preview
- [x] Docker deployment with auto-HTTPS

## Phase 2 - Advanced Features
- [ ] Spam detection (repeated messages, banned words)
- [ ] Command aliases and chained commands
- [ ] Custom API variables (fetch from external APIs)
- [ ] User points/loyalty system
- [ ] Song request integration
- [ ] Chat logs with search

## Phase 3 - Multi-Channel & Roles
- [ ] Multi-channel support per user
- [ ] Role-based access control (channel editors)
- [ ] Per-user command cooldowns
- [ ] Whisper support
- [ ] EventSub webhooks (follows, subs, raids)
- [ ] Stream alerts (in-chat announcements)

## Phase 4 - Premium Features
- [ ] Custom overlays (browser sources)
- [ ] Twitch Predictions & Polls integration
- [ ] Channel point rewards automation
- [ ] Clip creation commands
- [ ] Analytics dashboard (chat activity, command usage trends)
- [ ] Plugin/extension system
- [ ] Backup & restore
